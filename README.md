# FUTURE_DS_03
# College Event Feedback Analysis (Power BI)

## Project Overview

This repository contains **Power BI dashboards and a Python (Jupyter Notebook) analysis** created to analyze **college event feedback** collected from participants. The goal is to convert raw feedback data into meaningful insights that help organizers understand participant satisfaction and improve future events.

The project files provided (`Task 1.pbix`, `Task 2.pbix`, `Task 3.ipynb`) focus on:

* Ratings analysis
* Participant feedback trends
* Overall event performance
* Exploratory data analysis using Python
